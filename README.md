# 毛绒伙伴 (Pet Gacha) - Vibe Coding 项目

> 一个基于纯前端技术（HTML/CSS/JS）开发的宠物抽卡+养成游戏，全程通过自然语言对话驱动开发。

## Vibe Coding 提示词记录

本项目完全通过 AI 对话驱动开发，以下是核心开发过程中的提示词记录：

```

```

- 首页：大量宠物（狗）在页面上游走/链状动画，像水族馆那种感觉

- 动画流畅，卡通风格

## Vercel + Supabase 部署

### 快速部署步骤

1. **Fork/推送到 GitHub**
2. **在 Vercel 导入项目**
   - Framework Preset: `Other`
3. **创建 Supabase 项目**
   - 执行 `supabase-setup.sql`
   - 获取 URL 和 Anon Key
4. **在 Vercel 添加环境变量**
   ```
   SUPABASE_URL=https://your-project.supabase.co
   SUPABASE_ANON_KEY=your-anon-key
   ```

详细部署指南参见 [DEPLOY.md](./DEPLOY.md)

## Vibe Coding 开发流程

本项目采用 **Vibe Coding** 方式开发，核心流程如下：

1. **需求描述**：用自然语言描述想要的功能
2. **AI 生成**：AI 理解需求并生成代码
3. **快速迭代**：通过对话不断调整和完善
4. **测试验证**：运行测试确保功能正常
5. **持续优化**：根据反馈继续改进

### 开发统计

- **对话轮次**：18+ 轮核心功能对话
- **代码文件**：10+ 个文件
- **功能模块**：14+ 个核心系统
- **开发周期**：持续迭代优化
